其实挺简单（复杂的我也搞不来），mqtt传输下mac地址，格式转换下发给wakeonlan模块就行。

打包成文件用这个命令
  pyinstaller -F
  
第一次运行会生成 WakeByMqtt.ini 文件，在这个文件中改号自己的信息即可


mqtt包像这样的就行
  mac，ip/广播地址（默认192.168.1.255），端口（默认9）  广播地址和端口号可以不填
  82-85-34-22-11-09,192.168.1.2,9
  82:85:34:22:11:09,192.168.1.2,9
